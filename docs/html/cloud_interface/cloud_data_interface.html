Group queries based on the DataTitle

Do queries that are grouped based on the DataTitle to be processed (only queries that contain the specified DataTitle can be processed with the Module:Function/2 call that is specified (with arguments [Processing, State] where Processing is the list of queries and State is anything specific to the function) all the processed queries are removed from the QueryList so that it may be processed in other data modules, if necessary.
