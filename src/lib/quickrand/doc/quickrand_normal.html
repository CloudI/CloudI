box_muller/2

box_muller(Mean::number(), StdDev::number()) -> {Result1::float(), Result2::float()}

Box-Muller transformation for generating Gaussian noise.

Has numerical stability problems when X1 is very close to zero. This is a serious problem for stochastic modelling that is generating millions of numbers. If the result is used with added noise (e.g., creating a process sleep value) then it isn't a problem.
