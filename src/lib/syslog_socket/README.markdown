`syslog_socket`: Erlang syslog Client Interface
===============================================

A minimal syslog client interface that supports both
[RFC3164](https://tools.ietf.org/html/rfc3164) and
[RFC5424](https://tools.ietf.org/html/rfc5424).

Build
-----

    rebar compile
